# Branding Input Checklist (Phase 1)

Provide these assets so branding can be finalized from placeholders:

## 1) Logo Assets
- Primary logo (SVG preferred).
- Monochrome logo variants (light/dark use).
- Minimum safe-area guidance.

## 2) Color Tokens
- Primary brand color.
- Secondary/accent color.
- Body text color.
- Muted text color.
- Border/background neutrals.
- Optional status colors (success/warning/error).

## 3) Typography
- Primary font family name and license status.
- Web-safe fallback stack.
- Heading/body weights and recommended sizes.

## 4) PDF Template References
- Cover/page header/footer examples.
- Mandatory legal/disclaimer block text.
- Preferred table and chart framing style.

## 5) Email Template References
- Approved content block patterns.
- CTA button style and copy tone.
- Spacing rules for mobile responsiveness.
- Any forbidden components/styles.

## 6) Optional Extra Tokens
- Corner radius scale.
- Shadow/elevation guidelines.
- Icon set reference.
